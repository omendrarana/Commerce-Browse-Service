Note: Create Secrets.json file on local machine first to get access on Commercetools sdk.

Step1: Right click on web api project
Step2: Click on Manage User Secrets
Step3: Add the Jons format data as per below structure.

{
  "Client": {
    "ClientId": "",
    "ClientSecret": "",
    "ProjectKey": "",
    "AuthorizationBaseAddress": "",
    "ApiBaseAddress": "",
    "Scope": ""
  },
  "MeClient": {
    "ClientId": "",
    "ClientSecret": "",
    "ProjectKey": "",
    "AuthorizationBaseAddress": "",
    "ApiBaseAddress": "",
    "Scope": ""
  }
}

Step4: Add the relevant credentials (ClientId; ClientSecret; ProjectKey; AuthorizationBaseAddress; ApiBaseAddress and Scope)
